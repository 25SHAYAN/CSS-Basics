defining color value

colors in css more often specifies in the following formats:

  1. a color keyword -like -"red" , "green" , "blue" ,"transparent" , etc.
  2. a hex value - like "#ff000" , "ff0000" ,etc..
  3. a n rgb values (red , green , blue) - like "rgb(255, 0 0 )"
